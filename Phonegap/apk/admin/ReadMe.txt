Pour l'apk cordova, lorsque vous êtes en mode "grille", il faut:
- Choisir le nombre de lignes,
- Choisir le nombre de colonnes,
- Repartir vos watchers dans la grille de select,
- Appuyer sur le bouton valider,
- Et enfin double clicker sur l'image ou la video que vous voulez envoyer. (le drop ne fonctionne pas dans cette configuration)

NB: Il ne faut pas avoir plus de watchers que de nombre de cases dans votre grille. 
Si vous rencontrez un problème rafraichissez votre admin et vos watchers.
